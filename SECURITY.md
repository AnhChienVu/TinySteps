# Security Policy

If you discover a security issue in TinySteps, please do not open a public issue with exploit details.

Instead:

1. Gather the steps to reproduce the issue
2. Describe the potential impact
3. Share the report privately with the project maintainer

Please include:

- affected area
- environment
- reproduction steps
- any logs or screenshots that help explain the issue

We will review the report and work on a fix as quickly as possible.
